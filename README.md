# Nitzanim-Projects
Projects developed in the Nitzanim program under my supervision, showcasing mentoring, collaboration, and software development practices.

This repository showcases software projects developed by students in the Nitzanim program under my guidance.

The projects demonstrate:
- Hands-on programming practice
- Collaborative development using Git and GitHub
- Problem-solving and software design skills

My role across these projects included mentoring students, guiding technical decisions, and supporting collaborative workflows.


### MazeArcade
A collaborative practice project developed by instructors to strengthen core programming concepts through game development.

**Collaboration highlights:**
- Team-based development using Git and GitHub
- Shared responsibility over game logic and features
- Iterative development and debugging


### Pac-Man
A team-based Pac-Man style game developed collaboratively by students using GitHub.

**Collaboration highlights:**
- Shared codebase with multiple contributors
- GitHub-based collaboration and version control

**My role:**
- Guided the overall project structure
- Mentored students throughout development
